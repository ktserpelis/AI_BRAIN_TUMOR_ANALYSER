# Brain Tumor Segmentation (BraTS) with U-Net

Αυτό το project αναπτύσσεται στα πλαίσια προσωπικής έρευνας πάνω στη Βιοϊατρική Τεχνολογία και τη Νευροχειρουργική Ογκολογία.

## Στόχος
Εντοπισμός και κατάτμηση (segmentation) όγκων Γλοιοβλαστώματος από πολυκαναλικές μαγνητικές τομογραφίες (T1, T1ce, T2, FLAIR).
